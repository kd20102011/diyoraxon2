# Diyoraxon Project

Ushbu loyiha HTML5 va CSS3 texnologiyalaridan foydalangan holda yaratilgan va moslashuvchan (responsive) veb-sahifa interfeysini taqdim etadi.

---

## 🛠 Ishlatilgan texnologiyalar (Tech Stack)

* HTML5 — Sahifa tuzilishi va semantik strukturasi uchun
* CSS3 — Dizayn, uslublar va vizual ko'rinish uchun

---

## 💻 Xususiyatlari

* To'g'ri va toza semantik HTML kodi
* Chiroyli va zamonaviy CSS dizayn
* Barcha turdagi qurilmalar uchun moslashuvchan ko'rinish
